### Requirements
- Ruby >= 2.3
- [RubyGems](https://rubygems.org/pages/download)
- Linux, Unix, or Mac OS X
- Git

### Development

Follow [official jekyll docs](https://jekyllrb.com/docs/usage/). 

In short:

 - ```bundle install```
 - ```jekyll serve --livereload```
 - code your changes
 - ```git commit -m 'my awesome changes' && git push```
